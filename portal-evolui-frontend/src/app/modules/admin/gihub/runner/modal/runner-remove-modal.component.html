Linux / macOS (bash)

Abra o terminal já dentro da pasta do runner (onde estão config.sh e, se existir, svc.sh). O bloco abaixo não inclui cd — pode colar e executar tudo de uma vez. 

{{ linuxRemoveScriptBlock }}

No Linux, o config.sh remove do GitHub recusa execução com sudo na shell (Must not run with sudo). Execute como o utilizador dono da pasta (ex.: o do serviço actions.runner): sudo -u meuuser ./config.sh remove --token '…'

Windows (PowerShell)

Abra o PowerShell já na pasta do runner (onde está config.cmd). O script tenta parar e remover o serviço usando .service; depois executa config.cmd remove. Sem linha cd — pode colar o bloco inteiro. 

{{ windowsRemoveScriptBlock }}

Se usar apenas CMD, pare e remova o serviço em services.msc (ou sc stop / sc delete com o nome em .service) e execute manualmente: config.cmd remove --token "…"
